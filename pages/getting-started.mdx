# Get Started

Run your first AI workload in the cloud in 5 minutes.

<Steps>

### Create an account

Go to **[lightning.ai/sign-up](https://lightning.ai/sign-up)** and create a free account.

<Callout type="info">
The free tier includes one always-on Studio, 100 GB of persistent storage, 30M model tokens per month, and free GPU hours once you verify.
</Callout>

### Open your first Studio

From the dashboard, click **New Studio**, or go directly to **[studio.lightning.ai](https://studio.lightning.ai)**.

Your Studio opens on free CPU. Switch to GPU anytime using the hardware selector in the top-right corner.

<Callout>
**What is a Studio?** A Studio is your cloud-based dev machine. It has persistent storage, your IDE in the browser (or local via SSH), and hardware you can swap in one click.
</Callout>

### Run your first example

Pick the path that matches what you brought with you:

<Cards>
  <Cards.Card icon={<LayoutTemplate />} title="Clone a template (recommended)" href="https://lightning.ai/svekars-org/templates" arrow>
    Browse featured templates, click Clone. Start here if you don't yet have a project to work on.
  </Cards.Card>
  <Cards.Card icon={<GitBranch />} title="Connect your GitHub repo" href="https://lightning.ai/docs/platform/build/ai-studio" arrow>
    Bring your own code. Lightning pulls it into a Studio with dependencies installed.
  </Cards.Card>
</Cards>

</Steps>
