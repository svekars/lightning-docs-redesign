# Get Started

Run your first AI workload in the cloud in 5 minutes.

<Steps>

### Create an account

Go to **[lightning.ai/sign-up](https://lightning.ai/sign-up)** and create a free account.

<Callout type="info">
The free tier includes one always-on Studio, 100 GB of persistent storage, 30M model tokens per month, and free GPU hours once you verify.
</Callout>

### Open your first Studio

From the dashboard, click **New Studio**, or go directly to **[studio.lightning.ai](https://studio.lightning.ai)**.

Your Studio opens on free CPU. Switch to GPU anytime using the hardware selector in the top-right corner.

<Callout>
**What is a Studio?** A Studio is your cloud-based dev machine. It has persistent storage, your IDE in the browser (or local via SSH), and hardware you can swap in one click.
</Callout>

### Run your first example

Pick the path that matches what you brought with you:

- **Prebuilt Template.** Best if you don't have a project yet. Clone a working template and run it in one click.
- **AI Copilot.** Use Copilot inside a Studio to build an agent or app with AI assistance.
- **GitHub Repo.** Clone a GitHub repo into a Studio and treat it like your laptop.
- **Local IDE (SSH).** Keep using your local IDE (VSCode, Cursor, PyCharm) over SSH while the Studio runs remotely.
- **GPU Notebook.** Launch JupyterLab in a Studio for interactive work.

  <Tabs.Tab>
    Bring your own code from GitHub. Below is a walkthrough with the `hello-studio` starter project.

    1. Clone any repo into a Studio (just like your laptop). For example:

    ```bash
    git clone https://github.com/Lightning-AI/hello-studio.git
    ```

    2. Setup env your environment by installing all the needed dependencies:

    ```bash
    cd hello-studio/rag-system
    uv pip install -r requirements.txt
    ```
    
    Installs persist across restarts. Use `pip` or `uv`, whatever you prefer.

    3. **Example:** Scrape data to create a knowledge base for the RAG system.
    Real systems re-run every few hours to keep the knowledge base fresh:

    ```bash
    python scraper.py --page-limit 20 --update-every-hrs 24
    ```

    4. **Example:** Start RAG server. In a separate terminal window, start the RAG server.
    You'll now have two live servers running on the Studio:

    ```bash
    python rag_server.py
    ```

    5. **Example:** Test the server. Test the RAG server to make sure it's working:

    ```bash
    python client.py "What is the Matrix?"
    ```
