# Lightning for personal use

Lightning is great for solo work: side projects, work projects you want a real cloud environment for, and exploring AI without managing infrastructure. (Consolidates 5 previously separate pages.)

## Side projects

Lightning lets you spin up a real GPU environment for evening or weekend work without paying for an idle instance. Free tier covers most exploration; pay-as-you-go covers bursts. Persistent storage means your project picks up exactly where you left it.

## Work projects

If you're working on a professional project (side hustle, freelance, indie product, etc.) Lightning gives you a serious dev environment without the AWS / GCP setup overhead. Multi-cloud GPU access, public ports, and persistent storage are built in.

## Students

Lightning is perfect for homework, group projects, and hackathons. See the dedicated [academia](./academia.md) page for student-specific guidance.
