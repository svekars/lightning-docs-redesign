# Lightning for enterprise

Lightning scales AI/ML across enterprise teams of any size. This page covers the common enterprise setups in one place. (Consolidates 6 previously separate pages.)

## Enterprise teams

Lightning is designed for organizations that need shared GPU access, centralized billing, governance, and audit logs without sacrificing developer experience.

## Frontier labs

Frontier labs pretrain and deploy some of the largest models on Lightning. Multi-node training, infiniband interconnects, on-prem and multi-cloud bursting are first-class.

## ML use cases

Enterprises organize ML use cases at the teamspace level. Each use case (recommendations, fraud, vision QC, etc.) gets its own teamspace with scoped data, compute, and billing.

## Model development

A single teamspace can hold the full model development lifecycle from data cleaning to demos. Studios, datasets, experiment tracking, model registry, and deployment all live together.

## Infrastructure and admin tools

Lightning gives infra and MLOps teams tools to support researchers without becoming a bottleneck. Provisioning, budgets, access policies, and usage analytics are surfaced through admin dashboards and the CLI.
